# DMUX
## (Development tMUX)
### new names definitely being considered

## Installation 
macOS 
```
brew tap zdcthomas/tools
brew install dmux
```

## Usage
* `dmux` alone will use `fzf` to open up a list of dirs in `~`.
* `dmux --help` for help
